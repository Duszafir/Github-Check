# GitHub Profile Viewer

Una web simple y moderna para ver la info de cualquier usuario de GitHub al instante. 
Muestra su avatar, nombre, bio, Twitter, estadísticas y los 5 repositorios más recientes, 
todo con un diseño limpio y fácil de leer.

### Qué hace
- Avatar con hover “Ver perfil”.
- Nombre y Twitter alineados juntos.
- Bio que solo aparece si el usuario tiene.
- Estadísticas: followers, following y repos.
- Últimos 5 repositorios con links directos.
- Avisos cuando se alcanza el límite de requests de la API.
- Diseño responsive y animaciones suaves.

### Cómo usar
1. Escribí el username de GitHub.
2. Presioná Buscar.
3. ¡Listo! Toda la info aparece en pantalla.
